Installation

Installation is probably the easiest part about DZAP.

  1. Download the DZAP zip file
  2. Move the following files from the downloaded zip to the root of your mission pbo
    • the entire addons directory
    • the entire overwrites directory
    • DZAP_config
  3. Find this line in your mission file init.sqf (warning if you have a custom compiles file, find that line instead of the one below!)
    • call compile preprocessFileLineNumbers "\z\addons\dayz_code\init\compiles.sqf";
  4. Place this line directly after the line you found
    • call compile preprocessFileLineNumbers "addons\dzap\init.sqf";
  5. edit DZAP_config.sqf to configure the included mods to your liking. (see configuration section for more info])
